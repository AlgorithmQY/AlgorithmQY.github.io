1.N-维数组 ndarray

  • 注意： 引入Numpy,语法：import numpy as np
  • 产生数组，对数组进行操作，Numpy产生的数组操作时，时对每一个元素都进行操作，
  • array.shape表示array的几维几维，用一个元组进行表示，array.dtype输出数组的类型，array.ndim表示该数组的尾数
  • ndarray 是一个各向同性的数组，所以它当中的每一个元素应该是具有相同的数据类型
